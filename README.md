# なんでもデスゲイズ化プロンプト LP

好きな画像と共通プロンプトを画像生成AIへ入力し、覆面の人間キャラクターへ変換する企画の紹介LPです。

## ローカル確認

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173` を開きます。

## 公開

GitHub Pages向けの静的サイトです。公開リポジトリの `main` ブランチ直下をPagesの配信元に設定してください。

## 素材

- `assets/character.jpg`: 案内役キャラクター
- `assets/before-*`: 入力画像
- `assets/after-*`: 生成結果
- `assets/*-source.*`: 原素材の保管用コピー

画像の権利と各AIサービスの利用規約を確認したうえで公開してください。
